
    - 1️⃣ What is the difference between var, let, and const?
    variables declared with var are global scoped, meaning they can be access from anywhere in the code and they can be reassigned as needed.
    variables declared with the let and const keyword are block scoped, meaning they can only be accessed from their own scope or function but variables with const can not be reassigned. 


    - 2️⃣ What is the spread operator (...)?
    the spread operator breaks down an iterable object into its individual elements. for example using the spread operator on a string breaks it down to all the single letters in it. 

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    the map method returns a new array of the same length performing some operation on each element of the array.
    the filter method returns a new array after checking each element of the array if they pass a test or not. the length of the new array may or may not be the same as the original array.
    the foreach method does not return anything but it does loop over each element of the array.

    - 4️⃣ What is an arrow function?
    an arrow function in javascript is a more concise way of writing a function where you can assign the function itself to a variable and later pass it on to another function if necessary. 


    - 5️⃣ What are template literals?
    template literals are a special way of declaring string variable with the backtick `` symbols that allows you to insert variables inside them.
